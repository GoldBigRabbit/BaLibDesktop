# Uma Library Desktop

A library of doujin articles of _Blue Archive_

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run electron:serve
```

### Compiles and minifies for production
```
npm run electron:build
```

### Lints and fixes files
```
npm run postinstall
```
